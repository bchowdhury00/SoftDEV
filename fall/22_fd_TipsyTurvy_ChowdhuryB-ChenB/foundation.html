To get started with Foundation, download the Foundation file from their webite: Foundation

You start with a file containing a begining index.html file and a css and javascript folder. In the css folder, you have a foundation.css file set with all the defaults that foundation provides, such as the grid structure and the basic form elements demonstrated below. You are also given an empty app.css file that you can use to override or add onto the defaults.

Foundation also allows for easily usable forms inside the grid


On top of these forms, foundation provides the backbone for far more complex forms and features that are accessible on their website by going to the Building Blocks page: Building Blocks
These extra forms can be accessed in two ways. You can use the Foundation CDI, which is easily downloadable from their website with ample tutorials on how to use it. Or you can copy the css code provided into your app.css folder, and then copy the html and refactor as necessary. A few examples of this is shown below.
